Instructions for Running the Application

Command-Line Application

1. Open a terminal.
2. Navigate to the directory containing ToDo.java.
3. Compile the Java file: `javac ToDo.java`
4. Run the application: `java ToDo`
5. type in "help" for the guide
6. now "add/delete" "complete/incomplete","view","exit" can manage whatever you want

JavaFX Application

1. Ensure you have JavaFX installed and configured in your project.
2. Compile the Java file containing the JavaFX application: `javac TaskManager.java`
3. Run the JavaFX application: `java TaskManager`
4. after "To-Do List Manager" pop up, you can make whatever changes and use the app

Additional Notes

The working directory is crucial; ensure the "data.txt" file is created and located in the working directory.
Data persists across sessions, allowing users to interact with tasks even after restarting the application.
